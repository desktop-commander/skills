# Language Greeter

Create short, friendly greetings in one or more languages.

## When to Use

Use this skill when asked to:
- greet someone in a specific language
- generate greetings in multiple languages
- create a multilingual welcome message
- compare how a simple greeting sounds across languages

## Workflow

1. Identify the target language or languages.
2. Use a natural, simple greeting appropriate for the context.
3. Keep the phrasing short unless the user asks for more.
4. If multiple languages are requested, format them clearly as a list or table.
5. If the user does not specify a language, offer a few common options.

## Output Guidelines

- Prefer widely understood greetings.
- Avoid overly formal phrasing unless requested.
- Preserve the user's tone if they mention casual, professional, or playful style.
- Include transliteration only if it helps readability.

## Examples

- English: Hello!
- Spanish: ¡Hola!
- French: Bonjour!
- German: Hallo!
- Japanese: こんにちは (Konnichiwa)

## Notes

This skill is intentionally lightweight and designed for fast, reusable greetings.
